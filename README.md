# Woodworking Pro - Free Website

A complete woodworking calculator website with multiple tools for angle calculations, board feet, segmented rings, and more.

## Features

### 6 Professional Calculators:
1. **Miter Angle Calculator** - Perfect angles for frames and corners
2. **Compound Miter Calculator** - For crown molding and angled cuts
3. **Board Feet Calculator** - Calculate lumber volume and cost
4. **Segmented Ring Calculator** - For bowls and circular projects
5. **Right Triangle Calculator** - Solve any right triangle
6. **Roof Rafter Calculator** - Calculate rafter lengths and angles

### Website Features:
- Fully responsive (works on mobile, tablet, desktop)
- Professional design with smooth animations
- No sign-up required
- No ads
- Fast and lightweight
- Works offline once loaded

## Free Hosting Options

### Option 1: GitHub Pages (RECOMMENDED - 100% Free Forever)

1. **Create a GitHub account** (if you don't have one)
   - Go to https://github.com
   - Sign up for free

2. **Create a new repository**
   - Click the "+" in the top right
   - Select "New repository"
   - Name it: `woodworking-pro` (or any name you want)
   - Make it Public
   - Click "Create repository"

3. **Upload your files**
   - Click "uploading an existing file"
   - Drag and drop these 3 files:
     - index.html
     - styles.css
     - script.js
   - Click "Commit changes"

4. **Enable GitHub Pages**
   - Go to Settings (top of the page)
   - Scroll down to "Pages" (left sidebar)
   - Under "Source", select "main" branch
   - Click Save
   - Your site will be live at: `https://yourusername.github.io/woodworking-pro`

**Benefits:**
- Completely free forever
- No ads
- Custom domain possible (optional, $12/year)
- Fast loading
- Automatic SSL (https)
